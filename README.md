# SpartaZep
스파르타 게임개발 Unity 입문 과제

## 🖥️ 프로젝트 소개
Unity를 이용해 ZEP 을 모작했습니다.
<br>

## 🕰️ 제작 기간
* 24.10.08일 - 24.10.09일

### ⚙️ 개발 환경
- `Unity 2022.3.17f1`
- `VisualStudio 2022`
- `C#`
- `.NET 8.0`

## 📌 주요 학습 기능
#### 캐릭터 제어
- InputSystem을 이용한 캐릭터 행동 제어
#### UI
- 이름 변경, 캐릭터 스프라이트 변경, 상호작용 대화, 유저 리스트
#### 애니메이션
- 애니메이션 상태 정의 및 처리
#### 타일맵
- 타일맵 제작
#### 유틸
- 오브젝트 풀링
- 스크립터블 오브젝트 데이터
